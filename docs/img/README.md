# Screenshots

Drop hero screenshots / GIFs here. The top-level `README.md` and each
per-app `README.md` reference these by relative path, so the filenames
are load-bearing — keep the names below.

## Expected files

| Filename                | Used by                  | Suggested size      |
| ----------------------- | ------------------------ | ------------------- |
| `mac-hero.webp`          | top-level README         | 1600×1000, 2x       |
| `ipad-hero.webp`         | top-level README         | 1600×1200, 2x       |
| `iphone-hero.webp`       | top-level README         | 1080×1920, 3x       |

Optional supplementary images you can also reference from the per-app
walkthroughs (no current README references them, but conventions to
follow if you add them):

| Filename                | Notes                                    |
| ----------------------- | ---------------------------------------- |
| `mac-menubar.webp`       | Right-click menu shot                    |
| `ipad-stage-manager.webp`| 320 pt floating widget over Safari       |
| `ipad-pip.webp`          | PiP float over another app               |
| `iphone-portrait.webp`   | Full-screen avatar                       |

Save as PNG (sharp) or animated WebP (if you need motion). Keep
images under ~1 MB each so `git clone` stays fast.
